Pengertian OOP

Object Oriented Programming (OOP) adalah pendekatan dalam pemrograman yang mengorganisasi kode ke dalam bentuk objek. Objek ini merepresentasikan sesuatu yang memiliki data serta perilaku sehingga program menjadi lebih terstruktur dan mudah dikembangkan.

Konsep-Konsep OOP

  • Class sebagai cetakan untuk membuat objek.
  • Object sebagai hasil nyata dari class.
  • Encapsulation untuk membatasi akses langsung ke data.
  • Inheritance untuk mewariskan sifat ke class lain.
  • Polymorphism untuk memungkinkan method memiliki banyak bentuk.

🔹 Class Sederhana (Car)

Contoh berikut menunjukkan pembuatan class sederhana bernama Car di Dart. Class ini memiliki atribut brand dan method drive().
// Nama : Nabilah Cahya Suci // NIM : 251410018 // Kelas : SI2A class Car { String? model; String? color; void start() { print('Car started!'); } void stop() { print('Car stopped!'); } } void main() { Car myCar = Car(); myCar.model = "Sedan"; myCar.color = "Merah"; myCar.start(); myCar.stop(); Car anotherCar = Car(); anotherCar.model = "SUV"; anotherCar.color = "Biru"; anotherCar.start(); anotherCar.stop(); print("Model mobil saya: ${myCar.model}, warnanya: ${myCar.color}"); print("Model mobil lain: ${anotherCar.model}, warnanya: ${anotherCar.color}"); }

🔹 Object Instantiation

Bagian ini memperlihatkan proses pembuatan objek dari class Car yang sudah dibuat. Kita membuat instance myCar dan memanggil method drive().
// Nama : Nabilah Cahya Suci // Nim : 251410018 // Kelas : Si2A class Car { String model; String color; Car(this.model, this.color); void start() { print("Mobil $model dengan warna $color dinyalakan."); } void stop() { print("Mobil $model dihentikan."); } } void main() { Car myCar = Car("Sedan", "Merah"); Car anotherCar = Car("SUV", "Biru"); myCar.start(); anotherCar.start(); }

🔹 Constructors

Constructor digunakan untuk memberikan nilai awal pada atribut ketika objek dibuat. Berikut contoh class Person dengan constructor.
// Nama : Nabilah Cahya Suci // NIM : 251410018 // Kelas : SI2A class Car { String? model; String? color; Car(this.model, this.color); void start() { print('Car started!'); } void stop() { print('Car stopped!'); } } void main() { Car myCar = Car('Toyota', 'Red'); print('Model car: ${myCar.model}, color: ${myCar.color}'); myCar.start(); }

🔹 Practice Class Mahasiswa

Latihan implementasi class Mahasiswa dengan atribut nama dan nim, dilengkapi method perkenalan().
// Nama : Nabilah Cahya Suci // NIM : 251410018 // Kelas : SI2A class Mahasiswa { String nama; String nim; double ipk; Mahasiswa(this.nama, this.nim, this.ipk) { if (ipk < 0.0 || ipk > 4.0) { throw ArgumentError('IPK harus berada dalam rentang 0.0 - 4.0'); } } void tampilkanData() { print('===== Data Mahasiswa ====='); print('Nama : $nama'); print('NIM : $nim'); print('IPK : $ipk'); print('=========================='); } @override String toString() => 'Mahasiswa(nama: $nama, nim: $nim, ipk: $ipk)'; } void main() { Mahasiswa mhs1 = Mahasiswa('Budi Santoso', '20230001', 3.75); Mahasiswa mhs2 = Mahasiswa('Siti Aminah', '20230002', 3.92); mhs1.tampilkanData(); mhs2.tampilkanData(); print(mhs1); }

🔹 Encapsulation

Encapsulation membatasi akses langsung ke data menggunakan private variable (ditandai underscore) dan getter/setter.
// Nama : Nabilah Cahya Suci // NIM : 251410018 // Kelas : SI2A class Person { String _name; int age; Person(this._name, this.age); void setName(String name) { _name = name; } String getName() { return _name; } } void main() { Person person = Person("Salma", 20); print(person.age); person.setName('Aqila'); print(person.getName()); }

🔹 This Keyword

Keyword this digunakan untuk merujuk pada atribut milik class agar tidak tertukar dengan parameter yang namanya sama.
// Nama : Nabilah Cahya Suci // NIM : 251410018 // Kelas : SI2A class Product { String nama; Product(this.nama); } void main() { var product1 = Product('Keyboard Gaming'); print('Produk dibuat: ${product1.nama}'); }
